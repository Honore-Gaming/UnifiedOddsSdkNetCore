IReplayManager Methods

The IReplayManager type exposes the following members.

Methods
 NameDescription
AddMessagesToReplayQueue Adds event {eventId} to the end of the replay queue
GetAvailableScenarios Gets a list of available replay scenarios
GetEventsInQueue Gets the events in replay queue.
GetReplayEventsInQueue Gets list of replay events in queue.
GetStatusOfReplay Return the status of player. Possible values are: player is playing, player is stopped, player was never playing.
RemoveEventFromReplayQueue Removes the event from replay queue
StartReplay(Int32, Int32, NullableInt32, NullableBoolean) Start replay the event from replay queue. Events are played in the order they were played in reality
StartReplay(Int32, Int32, NullableInt32, NullableBoolean, NullableBoolean) Start replay the event from replay queue. Events are played in the order they were played in reality
StartReplayScenario Starts playing a predefined scenario
StopAndClearReplay Stop the player if it is currently playing and clear the replay queue. If player is already stopped, the queue is cleared.
StopReplay Stop the player if it is currently playing. If player is already stopped, nothing will happen.
