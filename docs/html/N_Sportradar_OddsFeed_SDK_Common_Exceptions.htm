Classes
 ClassDescription
CacheItemNotFoundException An exception thrown by the SDK cache components when the requested key was not found in the cache
CommunicationException An exception thrown by the SDK when an error occurred while communicating with external source (Feed REST-ful API)
DeserializationException An exception thrown by the SDK when a deserialization of the xml received from the feed fails
FeedSdkException A base class for all exceptions thrown by the feed SDK
MappingException An exception thrown by the SDK when the entity received from the feed could not be mapped to entity used by the SDK
NameExpressionException An exception thrown by name expressions when the name cannot be build
NameGenerationException An exception when the name for a market or outcome could not be generated
ObjectNotFoundException An exception thrown when the requested object is not found
RecoveryInitiationException An exception thrown when the recovery operation cannot be initiated due to after param to far in the past
